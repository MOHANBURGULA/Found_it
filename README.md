------------# Lost & Found------------------

A full-stack Lost & Found application with authentication, item reporting, and admin features.

## 🔧 Features
- User signup & login
- JWT authentication
- Report lost items with images
- Protected backend API
- MongoDB Atlas integration

## 📦 Tech Stack
Frontend:
- React (Vite)
- Tailwind CSS

Backend:
- Node.js
- Express
- MongoDB

## 🚀 Run Locally

### Backend
```bash
cd backend
npm install
npm run dev
